# EvoLib - 项目启动微授权

- **项目名称**：EvoLib (Evolution Library System)
- **项目愿景**：实现一个简单可用的图书管理系统，支持登录、用户管理、图书管理、读者管理、借阅管理核心功能。
- **项目经理（PM）**：songfq
- **授权日期**：2026-07-09
- **项目范围（当前阶段）**：
  - 采用 **单体架构**（前后端分离：Vue/React + Spring Boot）
  - 数据库使用 **PostgreSQL**
  - 暂不引入缓存（Redis）、消息队列（MQ）、微服务
  - 暂不涉及容器化部署（Docker）
- **核心目标**：
  1. 完成图书 CRUD（增删改查）
  2. 完成读者注册/登录（JWT 认证）
  3. 完成借书/还书流程（含超期天数计算）
- **工期预估**：1 周（基于简单单体架构）
